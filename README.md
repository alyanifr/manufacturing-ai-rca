<h1 align="center">AI-Driven Manufacturing Process Optimization System</h1> 

<div align="center"><b><i>"Copilot" for Manufacturing Process Engineers</i></b></div>

## **Project Description:**
- Developed an AI-powered manufacturing intelligence platform for anomaly detection, root-cause analysis, and process optimization using Transformer models and XGBoost.
- Built predictive analytics pipelines for manufacturing yield forecasting and equipment deviation detection using multivariate sensor data.
- Designed NLP-based engineering log analysis system to automate 8D reporting and corrective action recommendation workflows.
- Implemented explainable AI dashboards using SHAP for process transparency and engineering decision support.
- Containerized and deployed ML services using Docker and FastAPI for scalable industrial AI workflows. 

## **Core Features:**
1. Process Anomaly Detection
   + Goal: Detecting yield drops, abnormal machine behavior, cycle time spikes and defect clusters.
2. AI Root Cause Analysis Engine
   + Goal: Incorporates NLP with tabular fusion, incident logs, maintenance notes and process parameters for analysis.
3. FMEA Risk Scoring Assistant
   + Goal: Automatically estimates severity, occurrence, detection, and Risk Priority Number (RPN).
4. AI-Generated 8D Reports
   + Goal: Generate root cause summaries, corrective actions, containment actions and lesson learned.
5. BKM Recommendation System
    + Goal: Use embeddings and semantic similarity to retrieve historical successful fixes, recommened process settings and sites comparison.
